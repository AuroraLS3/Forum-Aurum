# Forum Aurum

This is a project for 'Tietokantasovellus' course.

## Goals:

Create a functional forum web application

## Functionality

- Users
  - Login/Logout
  - Registration
  - User Groups (Many to many)
    - Guest
    - Approved
    - Moderator
    - Admin
- Areas
  - Only displayed to specific groups
- Threads (CRUD)
  - (Voting)
  - Comments (CRUD)
